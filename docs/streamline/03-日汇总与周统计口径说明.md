# 日汇总与周统计口径说明

---

## 一、daily_summary 字段说明

| 字段                      | 中文   |
| ----------------------- | ---- |
| daily_budget            | 当日预算 |
| intake_calories         | 摄入热量 |
| exercise_calories       | 运动消耗 |
| remain_calories         | 还可摄入 |
| actual_deficit_calories | 实际缺口 |
| budget_deficit_calories | 预算缺口 |
| protein_total_g         | 蛋白摄入 |
| protein_target_g        | 蛋白目标 |
| fat_total_g             | 脂肪摄入 |
| fat_target_g            | 脂肪目标 |
| carb_total_g            | 碳水摄入 |
| carb_target_g           | 碳水目标 |
| first_meal_time         | 首餐时间 |
| last_meal_time          | 末餐时间 |
| eating_window_hours     | 进食时长 |

---

## 二、核心计算

intake = Σ diet_record
exercise = Σ sport_record

remain = budget - intake + exercise

actual_deficit = TDEE + exercise - intake

---

## 三、进食窗口

eating_window = last_meal - first_meal

---

## 四、周统计

avg_intake = Σ intake / 天数
avg_exercise = Σ exercise / 天数
avg_deficit = Σ deficit / 天数

---

## 五、异常口径

无饮食：

* intake = 0

无运动：

* exercise = 0

无预算：

* 全部为 null

---

## 六、核心结论

👉 daily_summary = 唯一统计来源
👉 周统计 = daily_summary 聚合
