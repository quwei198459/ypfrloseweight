# 数据表字段与算法映射说明

---

## 一、核心分层

用户 → 档案 → 预算 → 行为 → 汇总

---

## 二、关键表职责

| 表                  | 作用       |
| ------------------ | -------- |
| lw_user            | 用户身份     |
| user_profile       | 档案（预算输入） |
| user_budget_config | 预算结果     |
| diet_record        | 饮食真源     |
| sport_record       | 运动真源     |
| meal_record        | 餐次汇总     |
| daily_summary      | 日统计      |
| user_weight_record | 体重历史     |

---

## 三、字段映射（核心）

### 1. 预算

user_profile → user_budget_config

---

### 2. 饮食

food → diet_record → meal_record → daily_summary

---

### 3. 运动

sport_item → sport_record → daily_summary

---

### 4. 体重

user_weight_record → 趋势
user_profile.current_weight → 当前值

---

## 四、重要字段关系

### current_weight vs weight_record

* current_weight：快照
* weight_record：历史

---

### food vs diet_record

* food：模板
* diet_record：真实数据

---

### sport_item vs sport_record

* sport_item：模板
* sport_record：真实数据

---

## 五、最终结论

所有统计统一来源：

👉 daily_summary

所有行为真源：

👉 diet_record / sport_record / weight_record
