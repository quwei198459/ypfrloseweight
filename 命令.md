第一步：启动后端
cd backend
mvn spring-boot:run
第二步：启动管理后台
cd admin-frontend
nvm use 20.12.2
npm run dev
第三步：启动小程序本地调试
cd frontend
nvm use 20.12.2
npm run dev:mp-weixin
然后微信开发者工具导入：

frontend/dist/dev/mp-weixin



后端打包
cd backend
mvn clean package -DskipTests


管理后台构建
cd admin-frontend
nvm use 20.12.2
npm run build
产物：

admin-frontend/dist
小程序正式构建
cd frontend
nvm use 20.12.2
npm run build:mp-weixin
产物：

frontend/dist/build/mp-weixin
然后用微信开发者工具打开并上传：

frontend/dist/build/mp-weixin

networksetup -getinfo Wi-Fi
